Для беспроводной сети Wi-Fi важным моментом является выбор беспроводного канала работы. Оптимальный номер канала выбирается автоматически на основе периодического анализа радиочастоты (радиоэфира). Обычно, это радиоканал, максимально удаленный от соседних беспроводных сетей, чтобы исключить конфликт по частотам, который может повлечь проблемы при использовании беспроводной сети. Но также вы можете самостоятельно просканировать сеть Wi-Fi и отобразить подробную информацию по каждой беспроводной сети, находящейся в радиусе действия вашего беспроводного адаптера или точки доступа.

Windows

WirelessNetView

WirelessNetView — это небольшая утилита, работающая в фоновом режиме и отслеживающая активность беспроводных сетей вокруг вас. Для каждой обнаруженной сети она отображает следующую информацию: SSID, последнее качество сигнала, среднее качество сигнала, счетчик обнаружений, алгоритм аутентификации, алгоритм шифрования, MAC-адрес, RSSI, частота канала, номер канала и многое другое.

  • Скачать WirelessNetView с nirsoft.net
